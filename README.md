# FXQL Parser

## Description

An application made with [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Documentation

The documentation for this project is auth generated using swagger to be accessed at ```{url}/api```


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
